Description du projet

Dans ce projet, dans un groupe de 6 personnes, on devais mettre en place un environnement de travail avec des outils de gestion de projet afin de faciliter le travail et les interactions au sein de l'équipe. Enfin, on devais clairement identifier les différentes phases de votre projet, représenter votre cycle de développement et identifier/formaliser les jalons. On a formaliser ceci avec un diagramme de Gantt.

Ensuite, lors de la modelisation, on a développer une application de bureau en Java. L’IHM de cette application a été réalisée à l’aide de JavaFX en suivant le patron de conception Modèle– Vue–Contrôleur. On a développé cette application à partir du squelette de code fourni. En particulier la pérennisation des données a été effectuée à l’aide d’une sérialisation dans un fichier, dont le code est fourni.

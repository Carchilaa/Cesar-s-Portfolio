Description du projet

Dans ce projet, on devait être capables d'installer dans una machine virtuelle, le systeme operative Debian 11 pour que on puisse ensuite l'utiliser pour le developpement. Pour ceci, on devait installer un JDK, GIT et Intellij IDEA Community.

Ensuite, on a du réalisé une carte mentale explicative pour que n'importe qui puisse installer le systeme operative Debian 11 dans son ordinateur.

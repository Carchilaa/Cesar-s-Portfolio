Description du projet

Réalisation d'un déploiement complet de l'application Leed, un agrégateur RSS minimaliste, dans le cadre du module de virtualisation avancée. Ce projet a été réalisé individuellement en 2024/2025.

Aspects techniques

Le projet a impliqué la conception d'une infrastructure multi-tiers avec Docker et Docker-Compose. J'ai également travaillé sur la création d'images Docker sur mesure en respectant les bonnes pratiques de packaging, la gestion de la persistance des données à l'aide de volumes, ainsi que la centralisation de la configuration dans un fichier .env. Les modes d'exécution (développement et production) ont été distingués pour adapter les environnements.
